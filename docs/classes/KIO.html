    Class KIO<S, E, A>

    A functional programming interface for kintone operations.

    Example

    // Basic usage
    const result = await KIO.getRecord({ app: 1, id: 1 })
      .andThen("record", (record) => KIO.updateRecord({ record: { ...record, title: "Updated" } }))
      .run(runner);

    Staticcommit

    • commit(): KIO<object, KError, void>

      Commits all pending changes to kintone. This is required to execute any add, update, or delete operations. Multiple write operations can be stacked and will be executed together when commit is called.

      Returns KIO<object, KError, void>

      An effect that will commit all pending changes

      Example

      // Stack multiple operations and commit them together
      const kio = KIO.getRecord({ app: 1, id: 1 })
        .andThen((record) => KIO.updateRecord({ record: { ...record, title: "Updated" } }))
        .andThen(() => KIO.addRecord({ app: 1, record: { title: "New Record" } }))
        .andThen(() => KIO.deleteRecord({ record: anotherRecord }))
        .andThen(() => KIO.commit());
      await kio.run(runner);
      • Defined in kio.ts:592
